# NotesStack
NotesStack – B.Tech Academic Resource Hub
I created this as a project .It is a web application designed to help B.Tech students at Babu Banarasi Das University (BBDU), Lucknow easily access semester-wise notes and previous year question papers.

✨ Features:
Organized notes and papers for all 8 semesters
Clean and responsive UI using HTML, CSS, JavaScript, and Bootstrap
Fast and mobile-friendly design hosted on Netlify
Built with a focus on accessibility and ease of use

🎯 Purpose:
To support fellow students by providing a centralized platform for academic resources, especially tailored for BBDU.
